# よくある質問（FAQ）

LuraSwitch2 に関するよくある質問とその回答をまとめています。

## 基本的な使い方

### Q. プレハブを配置しても動作しないのですが？

A. 以下を確認してください：

1. **Udon Sharp がインストールされているか**
   - VCC (VRChat Creator Companion) から UdonSharp をインストールしてください
   
2. **制御対象が設定されているか**
   - Inspector で Target Light、Target Audio Source などが設定されているか確認してください

3. **VRChat SDK がインポートされているか**
   - VRChat SDK3 - Worlds が必要です

### Q. Unity のPlay モードでは動作するのに、VRChat では動かない

A. 以下を確認してください：

1. **ワールドのビルドとアップロード**
   - 変更後、必ずワールドをビルドしてアップロードしてください

2. **Udon のコンパイルエラー**
   - Console にエラーが出ていないか確認してください

## パフォーマンス

### Q. Quest で動作しますか？

A. はい、LuraSwitch2 は Quest に対応しています。ただし、以下の点に注意してください：

- リアルタイムライトは Quest では負荷が高いため、ベイクドライトの使用を検討してください
- ミラーは Quest では使用できません
- オブジェクトの数や複雑さに注意してください

### Q. 軽量化のためのベストプラクティスは？

A. 以下の点に注意してください：

- **必要最小限のコンポーネントのみ使用**
  - 使わないプレハブは配置しないでください
  
- **同期の最適化**
  - 個人設定（音量、ミラーなど）は Sync を false に
  
- **ライトの最適化**
  - リアルタイムライトの数を減らし、ベイクドライトを活用

## カスタマイズ

### Q. スクリプトをカスタマイズできますか？

A. はい、LuraSwitch2 のスクリプトは自由に編集できます。

- スクリプトは `Assets/QuickBrown/LuraSwitch2/02_CORE` にあります
- UdonSharp の知識があれば、機能を拡張できます

### Q. 見た目（UI）を変更できますか？

A. はい、プレハブの UI パーツは通常の Unity オブジェクトです。

- マテリアルやテクスチャを変更できます
- 位置やサイズを調整できます
- 完全に独自の UI に置き換えることも可能です

## 同期（Sync）

### Q. Sync を true にするとどうなりますか？

A. プレイヤー間で状態が同期されます。

- **true の場合**: 誰かがスイッチを切り替えると、全員に反映されます
- **false の場合**: 各プレイヤーが個別に設定できます

### Q. どのコンポーネントを Sync すべきですか？

A. 用途によって異なりますが、一般的には：

**Sync = true が推奨**
- Switch_Light（ワールド全体の照明）
- Switch_Object（共有オブジェクト）
- Switch_Video（共有ビデオ）

**Sync = false が推奨**
- Switch_Mirror（個人のパフォーマンス設定）
- Slider_Audio（個人の音量設定）
- Slider_Mirror（個人の品質設定）

## トラブルシューティング

### Q. エラーメッセージが表示される

A. エラーの種類によって対処法が異なります：

**「UdonSharp が見つかりません」**
- VCC から UdonSharp をインストールしてください

**「VRChat SDK が見つかりません」**
- VCC から VRChat SDK3 - Worlds をインストールしてください

**「コンパイルエラー」**
- Console のエラーメッセージを確認し、指示に従ってください

### Q. スイッチをクリックしても反応しない

A. 以下を確認してください：

1. **Collider が設定されているか**
   - UI パーツに Collider コンポーネントがあるか確認

2. **Event System が存在するか**
   - シーンに Event System があることを確認

3. **VRC Station などの干渉**
   - 他のシステムと干渉していないか確認
