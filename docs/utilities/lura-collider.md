# LuraCollider

![LuraCollider](/img/LuraCollider.png)

スライダーから操作できる、高さを変更することのできるコライダーです。

## 特徴

### 🎬 アニメーション効果

高さを変更したときに、滑らかにアニメーションします。

単なるコライダーの切り替えではなく、視覚的にもわかりやすい演出付きです。

### 📐 簡単な配置

`ReferenceBox` を変形させるだけで、好きな位置にコライダーを配置できます。

## 使い方

### 1. コライダーの配置

`ReferenceBox` オブジェクトを：

- 移動
- スケール変更
- 回転

することで、コライダーの位置とサイズを調整します。

### 2. Slider_Collider との連携

LuraCollider は **Slider_Collider** と組み合わせて使用します。

1. `Slider_Collider.prefab` を配置
2. LuraCollider を Target に設定
3. スライダーで高さを調整

## 配置時の注意

:::warning 重要
コライダーに乗った状態でスライダーを操作する配置にすると、プレイヤーが動いてしまい操作ができなくなる可能性があります。

**スライダーはコライダーの範囲外に配置してください。**
:::

### 良い配置例

```
[スライダー]  ←コライダーの外

      [コライダーエリア]
      ┌──────────┐
      │          │
      │          │
      └──────────┘
```

### 悪い配置例

```
      [コライダーエリア]
      ┌──────────┐
      │ [スライダー] ← ダメ！
      │          │
      └──────────┘
```

## パラメータ

### ReferenceBox

- コライダーのサイズと位置を決定
- Transform を調整することで配置

### 高さ調整

- Slider_Collider で動的に高さを変更可能
- 最小高さ・最大高さを設定可能

## 用途例

### 可変高さの台

プレイヤーの身長に合わせて台の高さを調整：

```
用途: 撮影用の台
高さ: 0m 〜 2m
```

### 昇降式の床

イベント用の昇降する床：

```
用途: ステージの昇降
高さ: 地面 〜 3m
```

### アスレチック

難易度調整可能なアスレチック：

```
用途: ジャンプ台
高さ: 0.5m 〜 3m（難易度に応じて）
```

## ベストプラクティス

### スライダーの配置

- コライダーの範囲外に配置
- 壁や柱など、固定された場所に設置
- わかりやすい位置（看板の近くなど）

### サイズ設定

- ReferenceBox は実際に使いたいサイズより少し大きめに
- 余裕を持たせることで乗りやすくなる

## トラブルシューティング

### コライダーが動かない

- Slider_Collider のターゲット設定を確認
- ReferenceBox のサイズを確認

### 操作中に落ちる

- スライダーの配置を見直す
- コライダーの範囲外にスライダーを移動

### アニメーションがカクカク

- 高さ変更の速度を調整
- スライダーのスナップ設定を確認（10%ずつ）
